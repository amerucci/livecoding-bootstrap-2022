Aligner horizontalement des div dans le container


Tout d'abord créer le container
Ensuite ajouter une row. Par défaut row utilisera du display flex par conséquent cherchera à aligner les div enfants les unes à coté des autres
Ajouter justify-content-center afin que celles-ci soient centrées dans la row.
Attention toute fois il faut que les div enfant aient une taille, soit en spécifiant un col-* ou alors en spéciant une taille en px ou %

        <div class="container">
            <div class="row justify-content-center bg-dark" >
            <div class="col-3 bg-warning">test>/div>
            <div class="col-3 bg-warning">test>/div>
            <div class="col-3 bg-warning">test>/div>
             >/div>
        >/div>

Aligner verticalement des div dans le container


Pour aligner verticalement une div dans notre row il va falloir un élément important pour que cela fonctionne.
En effet vous allez devoir définir la hauter de votre row pour que vos div puissent se placer au milieu de votre parent.
Sur votre row n'oubliez pas d'ajouter la class align-items-center 

            <div class="container">
                <div class="row  align-items-center bg-dark" style="height:250px">
                     <div class="col-3 bg-warning">test>/div>
                     <div class="col-3 bg-warning">test>/div>
                     <div class="col-3 bg-warning">test>/div>
                 >/div>
            >/div>

Le display flex


la class row a pour propriété css flex.
Ce qui implique que les enfants directs de row vont etre soumis à ce flex
Toutfois si vous avez une div dans une div (une div petit enfant) alors celle ci n'y sera pas soumis
C'est pourquoi il vous faudra spécifier d-flex en guise de class dans le parent 

            <div class="container">
                <div class="row justify-content-center bg-dark">
                     <div class="col-3 bg-warning d-flex">
                         <div>sous test>/div>
                         <div>sous test>/div>
                     >/div>
                     <div class="col-3 bg-warning">test>/div>
                     <div class="col-3 bg-warning">test>/div>
                 >/div>
            >/div>
